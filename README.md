## Quick Setup

1. Add `@vueless/nuxt` dependency to your project.

```bash
npm i @vueless/nuxt
```

2. Add `@vueless/nuxt` to the `modules` section of `nuxt.config.{ts,js}`

```js
{
  modules: [
    '@vueless/nuxt'
  ]
}
```
